Detailed Description

CMSIS Cortex-M3 Device USB OTG peripheral Header File. This file contains the USB OTG peripheral register's definitions, bits definitions and memory mapping for STM32F1xx devices.

Author
MCD Application Team

This file contains:

  • Data structures and the address mapping for the USB OTG peripheral
  • The Peripheral's registers declarations and bits definition
  • Macros to access the peripheral's registers hardware
